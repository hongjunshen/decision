1. Open solution file in folder DecisionWebAPI in visual studio, it will load three projects DecisionWebbAPI, DecisionClient and DecisionWebAPI. 

2. DecisionWebbAPI project used entity framework code first to generated database and database will be regenerated on each run.

3. in DecisionWebApi project Controllers folder file AccountController.cs, HomeController.cs and ValuesController.cs are auto generated, 

please ignore when examing.

4. in DecisionWebApi project Models folder AccountBindingModels.cs, AccountViewModels.cs and IdentityModels.cs are auto generated, 

when examing please ignore. 

5. solution is setup as multiple run (DecisionWebbAPI and DecisionClient). Please open solution file and click start to run 

database will be automatically generated.
